ABOUT THE PROJECT

MalGuard AI is a cutting-edge research initiative designed to tackle the growing threat of polymorphic and Zero-Day malware. Traditional antivirus software relies on signature matching, which fails against new, unknown threats.

MalGuard utilizes Static Analysis Machine Learning. Instead of running the potentially dangerous file (Dynamic Analysis), we extract mathematical features from the file's structure—such as its byte entropy, imported functions, and header metadata—to predict malicious intent with high confidence.

SYSTEM ARCHITECTURE

The MalGuard pipeline is a high-throughput, low-latency system built on Python Flask and LightGBM.

01
DATA INGESTION

The user uploads a binary via the web dashboard. The file is temporarily stored in a secure buffer. The LIEF library parses the PE (Portable Executable) format to access raw structural data.

02
VECTORIZATION

Raw data is transformed into a Feature Vector (NumPy Array). This involves hashing imported functions, calculating byte histograms, and normalizing entropy scores to match the EMBER topology.

03
AI INFERENCE

The LightGBM model (Gradient Boosting Machine) analyzes the feature vector. It assigns a probability score (0.0 to 1.0) indicating the likelihood of malicious intent.
